# SDR-Based-AoA-Direction-Finding-for-Drone-RF-Signals
This repository contains the GNU Radio Companion (GRC) flowgraphs and supporting files developed for a capstone project on Angle of Arrival (AoA) estimation for SDR-based drone direction finding.  The project investigates the use of Bartlett, MVDR, and MUSIC algorithms using a USRP B210 and GNU Radio.
